**शब्दार्थ** 

**ब्रूहि—** **कृपा करके हमें बतायें** **; कारणम्—** **कारण** **; एतस्य—** **इसका** **; दुर्धर्षत्वस्य—** **दुर्धर्षता का** **; मत्-रिपो:—** **मेरे शत्रु का** **;** **ओज:—** **पराक्रम** **; सह:—** **शक्ति** **; बलम्—** **बल** **; तेज:—** **प्रभाव** **; यत:—** **जहाँ से** **; एतत्—** **यह सब** **; समुद्यम:—** **प्रयास।** **.**

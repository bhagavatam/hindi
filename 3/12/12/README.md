**शब्दार्थ** 

मन्यु:, मनु:, महिनस:, महान्, शिव:, ऋतध्वज:, उग्ररेता:, भव:, काल:, वामदेव:, धृतव्रत: **—** **ये सभी रुद्र के नाम हैं।** **.**
